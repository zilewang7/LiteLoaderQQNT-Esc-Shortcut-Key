# LiteLoaderQQNT Esc Shortcut Key

在 QQNT 中使用 esc 来返回上一级 (退出当前聊天)


其他窗口按下 esc 来关闭， 主窗口按下 esc 返回主页面，双击 esc 关闭主窗口
